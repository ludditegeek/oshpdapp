Embedding code snippets within a web page

Developers often want to include code snippets on webpages and there are a few hacks out there to do this (some are pretty ugly - like creating an image file of the code). Others have server side dependencies. The example above uses the SyntaxHighlighter javascript library. This approach is pure javascript and has the advantage that the code can be pasted to the clipboard and then copied. To use this on your pages there are 3 steps:

1) Install the library code on your host and in your page create a references for the language of your choice (In my case I have installed the code in an 'includes' directory on my web server).
